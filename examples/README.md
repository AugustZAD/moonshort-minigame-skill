# Examples

This repository currently includes two migrated examples from the original moonshort project:

- `examples/platform-runner/index.html`
  - Side-scrolling runner/platformer example adapted from the previous `game/index.html`
- `examples/merge-2048/index.html`
  - Tile merge puzzle adapted from the previous `game/2048.html`

These examples are reference implementations for:
- bridge communication
- settlement screen structure
- mobile canvas sizing
- theme injection through `attribute` and `primaryColor`
